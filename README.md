# GrabGemini - Bulk Image Downloader

[![GitHub Repository](https://img.shields.io/badge/GitHub-View_Source-1a73e8?logo=github)](https://github.com/brian-caylor/GrabGemini_Image_Chrome_Extension)

A Chrome extension for bulk selecting and downloading high-resolution images directly from the Gemini "My Stuff" gallery at [gemini.google.com](https://gemini.google.com).

## Features

- **Checkbox selection** — each image in the gallery gets a checkbox overlay for easy picking
- **Select All** — select every image currently visible on screen
- **Lasso select** — click and drag to select multiple images at once
- **Bulk download** — downloads all selected images in one go as high-res master files
